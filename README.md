
## 🪑 ArrivingSoonToYou – Client Landing Page

[![Live Demo](https://img.shields.io/badge/Live-Demo-blue?style=for-the-badge)](https://arrivingsoontoyou.onrender.com/)

A minimal, modern, and responsive **launch landing page** built for a client promoting a curated collection of world-class furniture for the evolving future of work.

---

### 🚀 Overview

> “It’s not just furniture. It’s a commitment. A partnership. A movement.”

This static site was designed as a clean, high-impact **“Launching Soon”** experience. The page introduces the brand’s mission and visual identity, with strong typography, responsive design, and subtle visuals — optimized for performance and clarity.

---

### 💼 Built For

* A private client project — branding and structure customized to fit brand tone and message.

### 🧱 Tech Stack

* **HTML5** – semantic structure
* **CSS3** – custom responsive styling with modern design system
* **Google Fonts** – [Noto Sans](https://fonts.google.com/specimen/Noto+Sans)
* **Font Awesome** – social icons
* **Responsive Design** – fully mobile-friendly
* **Deployed on Render** – fast and free static hosting
